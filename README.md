# ✈️ Globetrotter - Friends Travel Leaderboard

Globetrotter is a fun, interactive, and visually stunning web application for you and your friends to track the countries you've visited, earn badges, and compete to see who is the ultimate explorer!

## 🌟 Features

- **🏆 Explorer Leaderboard**: See real-time ranks based on countries visited. Top 3 explorers get podium places (🥇, 🥈, 🥉).
- **🗺️ Interactive World Map**: Click directly on countries on the SVG world map to mark them as visited. Hover over any country to see who in your group has visited it.
- **🎖️ Travel Profiles & Badges**: Unlock achievements like *Euro Explorer*, *Island Hopper*, and *Around the World* as you log more countries.
- **🔌 Multi-device Sync via Supabase**: Seamlessly sync data with your friends in real-time, or use offline-first local storage.

## 🚀 Getting Started

### Option A: Local Storage (Offline Mode)
No configuration required! Just open `index.html` in any modern web browser. Your data is stored locally in your browser.

### Option B: Real-Time Sync (Supabase Mode)
To synchronize visits between multiple friends' devices:

1. Create a free account and project at [Supabase](https://supabase.com/).
2. Open the **SQL Editor** in your Supabase dashboard and run the following setup script to create the tables:

```sql
-- Create Friends Table
create table public.friends (
  id uuid default gen_random_uuid() primary key,
  name text not null unique,
  avatar text not null, -- Emoji avatar
  created_at timestamp with time zone default timezone('utc'::text, now()) not null
);

-- Create Visited Countries Table
create table public.visited_countries (
  id bigint generated by default as identity primary key,
  friend_id uuid references public.friends(id) on delete cascade not null,
  country_code varchar(3) not null, -- ISO 2-letter country code
  created_at timestamp with time zone default timezone('utc'::text, now()) not null,
  unique (friend_id, country_code)
);

-- Enable Realtime for live updates
alter publication supabase_realtime add table public.friends;
alter publication supabase_realtime add table public.visited_countries;
```

3. Run the app locally (e.g. using a local server or directly opening `index.html`).
4. Click the **🔌 Supabase Setup** button in the top right corner.
5. Enter your **Supabase Project URL** and **Anon API Key** (found in your Supabase Project Settings under API).
6. Click **Connect & Sync**. The application will now read and write to your Supabase cloud database in real-time!

## 🛠️ Technology Stack

- **Frontend**: Vanilla HTML5, CSS3 (Glassmorphism, custom dark theme, custom responsive grid), Vanilla JavaScript (ES6).
- **Database**: Supabase (PostgreSQL database & real-time listeners).
- **Map Representation**: Inline-manipulated vector SVG world map mapping country paths to ISO 3166-1 alpha-2 codes.
